## Githubへのコミットメッセージのルール

- 必ず日本語で記述してください。
- 1行目は50文字以内にしてください。
- 絶対パスや機密情報をコメントに記述しない。
- 以下のいずれかのプレフィックスを使用してください:
  - feat: 新機能の追加
  - fix: バグ修正
  - docs: ドキュメントの変更
  - refactor: リファクタリング
  - style: フォーマットの変更
